# Cornerstone Economics typography

## Primary typeface

**Avenir Roman** — inherited from the 2018 brand standard set by Lay.

Use for: wordmark, headings, body copy in formal deliverables (reports, proposals, letterhead, presentations), signature blocks.

## Fallback stack

For web and email, where Avenir is not universally available:

```
font-family: "Avenir Roman", "Avenir", "Avenir Next", Arial, sans-serif;
```

In HTML email signatures, use `Arial` as the rendered fallback since most clients do not render custom fonts. The stack above ensures the best match where possible without visual glitches where not.

## Weights

- Regular (400) for body copy
- Semibold / Medium for emphasis within body
- Bold for headings and the wordmark

Avoid italic except for taglines or quoted material.

## Taglines

- Primary positioning: **"Building resilience, transforming society."**
- Legacy descriptors (use sparingly, context-dependent): *Research — Strategy — Insights — Solutions*
