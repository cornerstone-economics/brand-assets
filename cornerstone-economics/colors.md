# Cornerstone Economics color palette

Derived from the current CE logo. Hex values are approximations from the rendered PNG and should be refined against the vector source when available.

## Primary

| Name | Hex | Usage |
|---|---|---|
| Steel blue | `#4A7A9A` | Wordmark, headings, signature text, chart primary series, body accents on white |

## Accents

| Name | Hex | Usage |
|---|---|---|
| Coral | `#E76F51` | Single-point emphasis, callouts, secondary chart series. Use sparingly. |
| Gold | `#E8B84D` | Tertiary accent, multi-series chart fills, infographic highlights |
| Peach | `#F4BC8B` | Soft accent, background washes, supporting chart fills |
| Beige | `#C9B99B` | Neutral warm, supporting chart fills, dividers |
| Powder blue | `#B8D0D8` | Cool neutral, background washes, light accents |

## Usage rules

- Body text: near-black (`#091F38` or `#333`) on white. Never accent colors on white for body text (contrast fails).
- Headings and UI accents: primary steel blue.
- Charts: start with steel blue + coral as the two-series default. Add gold, beige, powder blue for additional series.
- Backgrounds: white or powder blue wash. Avoid saturated accent backgrounds.
- Signature text: steel blue on white, following the DCDC-style template convention.
