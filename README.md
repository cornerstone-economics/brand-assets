# Brand assets

Official brand assets for Cornerstone Economics, a trade name of Capstone Research Group VBA.

## Structure

- `cornerstone-economics/` — logo, colors, typography
- `signatures/` — per-person HTML email signatures

## Usage

- **Logos:** use the PNG in `cornerstone-economics/logo/`. Reference via `raw.githubusercontent.com` for embedding in email signatures and external materials.
- **Colors and typography:** see `colors.md` and `typography.md`.
- **Email signatures:** each principal has their own HTML file under `signatures/`. Copy the file contents and paste into the HTML signature editor of your email client.

## Updating

Commit new versions with a short note in the commit message. Supersede old versions rather than deleting, so historical references don't break.
